    Goddard with state constraints

    using CTProblems

    You can access the problem in the CTProblems package:

    prob = Problem(:goddard, :classical)
    title           = Goddard problem with state constraint - maximise altitude
model    (Type) = CTBase.OptimalControlModel{CTBase.Autonomous, CTBase.NonFixed}
solution (Type) = CTBase.OptimalControlSolution

    Then, the model is given by

    prob.model
    
tf ∈ R, variable
t ∈ [t0, tf], time
x ∈ R³, state
u ∈ R, control
x(t0) == x0, initial_con
r = x₁
v = x₂
m = x₃
m(tf) == mf, final_con
0 ≤ u(t) ≤ 1, u_con
r0 ≤ r(t) ≤ Inf, x_con_rmin
0 ≤ v(t) ≤ vmax, x_con_vmax
ẋ(t) == f(x(t), u(t))
r(tf) → max

    You can plot the solution.

    plot(prob.solution, size=(700, 900))
