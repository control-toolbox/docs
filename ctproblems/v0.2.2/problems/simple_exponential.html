    Simple exponential

    using CTProblems

    The energy-min simple exponential problem consists in minimising

    \[ \frac{1}{2}\int_{0}^{1} u^2(t) \, \mathrm{d}t\]

    subject to the constraints

    \[ \dot x(t) = - x(t) + u(t),\]

    and the limit conditions

    \[ x(0) = -1, \quad x(1) = 0.\]

    You can access the problem in the CTProblems package:

    prob = Problem(:exponential, :dim1, :energy)
    description     = simple exponential - energy min
model    (Type) = CTBase.OptimalControlModel{:autonomous, :scalar}
solution (Type) = CTBase.OptimalControlSolution

    Then, the model is given by

    prob.model
    Optimal control problem of the form:

    minimize  J(x, u) = ∫ f⁰(x(t), u(t)) dt, over [t0, tf]

    subject to

        ẋ(t) = f(x(t), u(t)), t in [t0, tf] a.e.,

        ϕl ≤ ϕ(x(t0), x(tf)) ≤ ϕu, 

    where x(t) ∈ R and u(t) ∈ R.

    Besides, t0, tf are fixed.

    You can plot the solution.

    plot(prob.solution, size=(700, 400))
